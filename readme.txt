Form Select2
-------------------------------------------------------------------------------

Version: 1.0.0
Author: Guillem Lorman <guillem@bajoelcocotero.com>
Build Date: 23 June 2012
Github Repository: 
Requirements: Symphony 2.3

Apply Select2 plugin in Symphony CMS backend forms. Select2 is a jQuery based replacement for select boxes. It supports searching, remote data sets, and infinite scrolling of results.
 
jQuery Select2 Plugin:  
http://ivaynberg.github.com/select2/

Installation
-------------------------------------------------------------------------------

1. Upload the 'form_select2' folder in this archive to your Symphony
   'extensions' folder.

2. Enable it by selecting the "Form Select2", choose Enable from the
   with-selected menu, then click Apply.

3. Now the Select Boxes of your backend forms will be replaced for the Select2 verion.
